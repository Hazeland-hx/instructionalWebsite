    Skill2: Create a quote

    Just add a '> ' before the quoted text. References can also be nested, such as adding '>>' or '>>>' or even n '>'s...

    Example

    > Quote1 >>Quote2 >>>>>>>>>>QuoteN

    Learn by Doing

    Skill3: Insert a code

    Single line of code

    Each code is wrapped with a ' ':

    `code here`

    Example

    `create database hero;`

    Code block

    JThe codes are wrapped with three **'** s respectively, and the backquotes on both sides occupy a separate line.

    (```) code here1... code here2... code here3... (```)

    Example

    (```) function fun(){ echo "This is a great line of codes!"; } fun(); (```)

    Learn by Doing
