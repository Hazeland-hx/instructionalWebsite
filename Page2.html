    What is Markdown?

    Markdown is a markup language that can be written with a common text editor.

    Through a simple markup syntax, it can give common text content a certain format.

    Learn by Doing

    C. It is a smart keyboard tool that help you to take notes quickly.

    Why Markdown?

    Markdown's syntax is concise and clear, easy to learn, and more powerful than plain text, so many people use it to write blogs. The world's most popular blogging platform WordPress and large-scale CMS such as Joomla and Drupal can support Markdown well. And execpt for writing blogs, Markdown Language is also a great tool to take notes as it allows students to adjust the layout and insert things easily. And we are going to see some applications in this course.

    Which platform should I choose to use Markdown?
