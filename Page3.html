Skill2: Create Different text highlights using Markdown

Highlighting some texts is quite common for people when taking notes. The usage in Markdown language is quite simple: ** or -- stands for bold, * or - stands for italic, ~~ stands for delete.

**bold** or __bold__

*italic* or _italic_

~~delete~~

Skill4: Insert Pictures and Links using Markdown

Pictures and links are also indispensable in taking notes!

The syntax of inserting pictures and links are quite similar, the only difference is the "!". The formats of these two are as follows.

Pictures：![]() ![Text of the picture(optional)](Address of the picture)

Links：[]() [Text of the link](Address of the link)

And links are divided into inline links, referential links and automatic links.

Inline Link：[Hazel's website](http://www.hazeland.xyz) Referential Link：[Hazel's website][url]，the url is a link mark, which can be placed anywhere in the text. [url]:http://www.hazeland.xyz The format of the link tag is: [link tag text]: link address link title (optional) Automatic link( which is enclosed with `<>`): Picture: ![Me](https://i.pinimg.com/originals/f2/20/b6/f220b6f46c57ac6583e091597f53a7e1.jpg)
